Добро пожаловать!
=================

Русскоязычное сообщество Write the Docs объединяет всех, кто создаёт и использует техническую документацию.

Это часть международного сообщества `Write the Docs <wtd_>`_.

..  _wtd: http://www.writethedocs.org/


Общение
-------

Присоединяйтесь к нам в `Telegram <telegram_>`_ и чате #russia в `Слаке Write the Docs <wtd-slack_>`_.

Вступайте в группы митапов в `Санкт-Петербурге <meetup-spb_>`_ и `Новосибирске <meetup-novosibirsk_>`_.

..  _meetup-spb: https://www.meetup.com/Write-the-Docs-Russia/
..  _meetup-novosibirsk: https://www.meetup.com/Write-the-Docs-Siberia/

..  _telegram: https://t.me/technicalwriters
..  _wtd-slack: http://slack.writethedocs.org/
